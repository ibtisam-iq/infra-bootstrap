# Helm

K8s package manager for charts/templates. Installs CLI for deploying apps.

## Installation Command

```bash
curl https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-4 | bash
```

## What It Installs

- Helm CLI (latest v4).

## Verify

```bash
helm version  
```
