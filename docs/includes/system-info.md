!!! example "System Information Collection"
    This component gathers a detailed snapshot of the host system, including:

    - Hostname
    - Private and public IP addresses
    - MAC addresses for all active interfaces
    - Network interface assignments
    - DNS resolver configuration
    - Kernel version
    - OS release and codename
    - CPU model and architecture
    - Total memory
    - Disk usage summary
    - CPU load averages
    - Machine UUID

    This information helps verify whether the host meets required thresholds for container runtimes and Kubernetes components.
